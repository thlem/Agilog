# Agilog
A simple angular/node app

## Installation
 - npm install
  - (if brunch is not installed globally)
   - npm install -g brunch
  - (if bower is not installed globally)
   - npm install -g bower
 - bower install
 - brunch build
 - nodemon
 - You can now access at http://localhost:3131
